# 🧱 백엔드 도장깨기 스터디

백엔드 개발자로서 필수적인 기능들을 하나씩 직접 구현하며 기록하는 스터디입니다.  
단순 구현에 그치지 않고, **왜 그렇게 만들었는지**, **어떤 부분에서 고민했는지**도 함께 정리하고자 합니다.
